# Chamber Visualizer

A browser-based tool to create animated GIFs visualizing air chamber sequences — for compression garments, massage devices, or similar products.

## Features
- 4–12 chambers configurable
- Timeline editor: click or drag to activate/deactivate time steps per chamber
- Presets: Sequential, Parallel, Wave, Alternating, Center-out
- Adjustable pill appearance: width, height ratio, gap, tilt, padding
- Transparent background support
- Loop modes: Loop, Once, Ping-Pong
- Export as animated GIF at configurable output size

## Usage
Open `index.html` in any modern browser. No build step or installation required.

Available via GitHub Pages: https://wellcosan-gmbh.github.io/chamber-visualizer

## Tech
- Vanilla JS + Canvas API
- gif.js for GIF encoding (loaded via CDN)
